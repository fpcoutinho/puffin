# Puffin

Este projeto é uma aplicação web que consome a API rest Narwhal, uma automatizadora do processo de geração laudos de engenheiros eletricistas.

| ✨ Nome        | **Puffin** |
| 🏷 Tecnologias | vue, vite  |
